# 📖 Blog Break

A modern, creative, and responsive blog application built with **Vanilla JS**, **HTML/CSS**, and a **C# ASP.NET Core API** powered by **SQLite**.

## ✨ Features
- **Modern Design**: High-end editorial aesthetic with the "Blog Break" brand.
- **Editorial Experience**: Sophisticated typography and polished interactive elements.
- **Hidden Admin Panel**: Secure "Logo-Trigger" access for the site owner.
- **Premium Social Sharing**: Interactive, stylized sharing buttons with brand-accurate colors.
- **SQLite Powered**: A portable, file-based database perfect for free hosting.
- **Responsive Management**: Full mobile-friendly admin dashboard for creating and managing stories.

## 🚀 Getting Started

### Prerequisites
- .NET 10 SDK
- Node.js (for optional local dev server)

### Local Development
1.  **Backend**:
    ```bash
    cd BlogApi
    dotnet run
    ```
    The API will start at `http://localhost:5041`.
2.  **Frontend**:
    Open `frontend/index.html` in your browser (use Live Server for the best experience).
